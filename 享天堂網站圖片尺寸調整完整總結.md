## 🔧 調整方式詳細說明

### 1. 修改工具和流程

#### 使用的主要工具：
- **搜尋工具**: `grep` 指令查找特定的圖片比例設定
- **檔案編輯**: VS Code 的 `replace_string_in_file` 功能
- **驗證工具**: 終端機指令驗證修改結果

#### 標準調整流程：
1. 📋 **分析規格**: 根據規格表確定目標比例
2. 🔍 **搜尋現狀**: 使用 `grep` 找出所有相關設定
3. ✏️ **精準修改**: 使用具體的上下文進行字串替換
4. ✅ **驗證結果**: 確認修改後的設定符合規格
5. 📝 **記錄文件**: 建立詳細的修改報告

### 2. 常用的調整模式

#### 固定高度 → 響應式比例：
```diff
- <div class="w-full h-64 overflow-hidden">
+ <div class="w-full aspect-[4/3] overflow-hidden">
```

#### 複雜比例 → 標準比例：
```diff
- aspect-[16/9] md:aspect-[21/9] lg:aspect-[2.4/1]
+ aspect-[4/3] md:aspect-[16/9]
```

#### 非標準比例 → 4:3 標準比例：
```diff
- aspect-[800/544]
+ aspect-[4/3]
```

#### 圖片來源修正：
```diff
- service.cover?.desktop
+ service.background?.desktop
```

### 3. 驗證指令範例

#### 搜尋所有 aspect 設定：
```bash
grep -r "aspect-" src/
```

#### 檢查特定頁面的圖片設定：
```bash
grep -n "aspect-\[.*\]" src/components/HeroBanner.astro
```

#### 驗證修改結果：
```bash
grep -n "aspect-\[.*\]\|h-64" src/routes/joinUs/index.astro
```

## 🎨 視覺改進效果

### 1. 標準化比例設定
- **Logo**: `3:1` - 統一品牌識別比例
- **Banner**: `16:9`（電腦版）/ `4:3`（手機版）
- **正方形圖片**: `1:1` - 團隊成員、本週推薦
- **一般內容圖**: `4:3` - 最新消息、服務項目、異業合作
- **服務項目首頁**: `2:1`（電腦版）/ `1:1`（手機版）

### 2. 響應式斷點策略
```css
/* 手機版（預設） */
aspect-[4/3]

/* 電腦版（md 以上，768px+） */
md:aspect-[16/9]

/* 大螢幕（lg 以上，1024px+） */
lg:aspect-[2/1]
```

### 3. Tailwind CSS 工具類別
#### 主要使用的 aspect-ratio 類別：
```css
aspect-square          /* 1:1 正方形 */
aspect-[4/3]          /* 4:3 標準比例 */
aspect-[16/9]         /* 16:9 寬螢幕比例 */
aspect-[2/1]          /* 2:1 寬版比例 */
aspect-[640/1280]     /* 9:18 手機版直立比例 */
aspect-[1920/1080]    /* 16:9 高解析度比例 */
```
