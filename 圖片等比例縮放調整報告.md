# 圖片等比例縮放調整報告

## 概述
將網站所有圖片的縮放方式從 `object-cover`（裁切模式）改為 `object-contain`（等比例模式），確保圖片完整顯示且不會被裁切，保持原始比例。

## 調整範圍
本次調整涵蓋了以下 11 個檔案，共修改了 18 個圖片元件：

### 1. 首頁相關組件

- **IntroSection.astro**: 品牌介紹區塊背景圖片
- **IndexNews.astro**: 首頁最新消息圖片（桌面版和手機版）

### 2. 品牌介紹頁面

### 3. 最新消息頁面

## 技術細節

### 修改內容
將所有圖片的 CSS class 從：
```css
object-cover
```
改為：
```css
object-contain
```

### 影響範圍
1. **視覺效果變更**：
   - 圖片不再被裁切，完整顯示
   - 保持原始寬高比例
   - 可能在容器內留有空白區域

2. **響應式設計**：
   - 同時調整了桌面版（`hidden md:block`）和手機版（`md:hidden`）圖片
   - 保持跨裝置的一致性體驗

3. **動畫效果保留**：
   - 維持原有的 hover 縮放效果（如 `hover:scale-110`）
   - 保留過渡動畫（如 `transition-transform duration-300`）
