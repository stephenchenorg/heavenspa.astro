# ServiceSection 統一 3:2 比例調整報告

## 調整日期
2025年11月3日

## 調整項目
ServiceSection.astro 圖片容器統一調整為 3:2 比例

## 變更內容

### 1. HTML 結構調整
**變更位置**：圖片容器 div 的 class 屬性
```html
<!-- 調整前 -->
<div class="service-image-container relative aspect-square lg:aspect-[3/2] w-full lg:w-[700px] overflow-hidden rounded-3xl shadow-2xl">

<!-- 調整後 -->
<div class="service-image-container relative aspect-[3/2] w-full lg:w-[700px] overflow-hidden rounded-3xl shadow-2xl">
```

### 2. 統一比例設定

#### 所有裝置統一
- **比例**：3:2 (1.5:1)
- **CSS 類別**：`aspect-[3/2]`
- **適用範圍**：所有螢幕尺寸
- **優點**：
  - 統一的視覺體驗
  - 更好的內容展示比例
  - 適合服務場景圖片

### 3. CSS 樣式全面更新

#### 3.1 移除響應式比例差異
```css
/* 調整前：分別處理 */
.aspect-square { /* 手機 1:1 */ }
.lg:aspect-[3/2] { /* 桌面 3:2 */ }

/* 調整後：統一處理 */
.aspect-[3/2] { /* 所有裝置 3:2 */ }
```

#### 3.2 圖片定位優化
```css
.aspect-[3\/2] .service-image-container img {
  /* 統一 3:2 容器：適合大部分服務場景圖 */
  object-position: center center;
}
```

#### 3.3 響應式微調
```css
/* 小螢幕：稍微偏向上方 */
@media (max-width: 480px) {
  object-position: center 40%;
}

/* 平板及以上：完全居中 */
@media (min-width: 481px) {
  object-position: center center;
}
```

### 4. 手機版 3:2 比例優勢

#### 4.1 視覺效果改善
- **更寬的視野**：比 1:1 提供更多水平空間
- **更好的場景展示**：適合展示服務環境
- **統一性**：所有裝置保持一致的視覺比例

#### 4.2 空間利用
- **垂直空間節省**：比 1:1 節省約 25% 高度
- **內容密度提升**：在有限螢幕上展示更多內容
- **瀏覽效率**：用戶可以更快瀏覽服務項目

### 5. 手機版適配優化

#### 5.1 觸控體驗
- **適當的寬高比**：3:2 在手機上仍保持良好的觸控區域
- **視覺舒適**：不會過於扁平或過於方正
- **操作友善**：輪播操作更加順暢
