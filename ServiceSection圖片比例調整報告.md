# ServiceSection 圖片比例調整報告

## 調整日期
2025年11月3日

## 調整項目
ServiceSection.astro 圖片容器長寬比例調整

## 變更內容

### 1. HTML 結構調整
**變更位置**：圖片容器 div 的 class 屬性
```html
<!-- 調整前 -->
<div class="service-image-container relative aspect-square lg:aspect-[2/1] w-full lg:w-[700px] overflow-hidden rounded-3xl shadow-2xl">

<!-- 調整後 -->
<div class="service-image-container relative aspect-square lg:aspect-[3/2] w-full lg:w-[700px] overflow-hidden rounded-3xl shadow-2xl">
```

### 2. 新的比例設定

#### 手機版 (Mobile)
- **比例**：1:1 (正方形)
- **CSS 類別**：`aspect-square`
- **適用範圍**：螢幕寬度 < 1024px
- **優點**：
  - 適合手機直向瀏覽
  - 統一的視覺呈現
  - 節省垂直空間

#### 桌面版 (Desktop)
- **比例**：3:2 (1.5:1)
- **CSS 類別**：`lg:aspect-[3/2]`
- **適用範圍**：螢幕寬度 ≥ 1024px
- **優點**：
  - 更適合服務場景展示
  - 平衡的視覺比例
  - 較 2:1 更緊湊，節省空間

### 3. CSS 樣式同步更新

#### 3.1 容器樣式
```css
.service-carousel-container .relative.lg\:aspect-\[3\/2\] {
  /* 3:2 比例容器優化 */
}
```

#### 3.2 圖片定位優化
```css
.lg\:aspect-\[3\/2\] .service-image-container img {
  /* 3:2 容器：適合服務場景和環境圖片 */
  object-position: center center;
}
```

#### 3.3 響應式適配
```css
.service-carousel-container .lg\:aspect-\[3\/2\] img {
  /* 3:2 容器：適合服務場景和環境圖片 */
  object-fit: cover;
  object-position: center center;
}
```

### 4. 視覺效果比較

| 裝置類型 | 調整前比例 | 調整後比例 | 視覺改進 |
|---------|-----------|-----------|----------|
| 手機版 | 1:1 | 1:1 | 保持不變 ✓ |
| 桌面版 | 2:1 | 3:2 | 更平衡緊湊 ✓ |

### 5. 優勢說明

#### 3:2 比例優勢
1. **更自然的視覺比例**：接近黃金比例，視覺感受更佳
2. **適合服務展示**：既不會太寬也不會太窄
3. **空間利用更佳**：比 2:1 節省 25% 的垂直空間
4. **通用性更強**：適合各種類型的服務圖片

#### 1:1 比例維持
1. **手機友善**：正方形在小螢幕上顯示效果最佳
2. **一致性**：所有服務卡片保持統一視覺
3. **觸控優化**：正方形更適合手機觸控操作

### 6. 技術實現細節

#### 6.1 Tailwind CSS 類別

#### 6.2 響應式斷點
- **手機**: < 1024px → 1:1 比例
- **桌面**: ≥ 1024px → 3:2 比例
