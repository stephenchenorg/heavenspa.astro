# 圖片智能顯示策略調整報告

## 概述
將網站所有主要圖片改為「智能顯示策略」：**優先等比例縮放，必要時置中裁切**的做法，確保最佳的視覺效果和使用者體驗。

## 🎯 調整策略

### 新的圖片顯示邏輯：
```css
object-cover object-center
```

**工作原理**：
1. **優先嘗試**：圖片盡可能完整顯示
2. **智能裁切**：當比例不匹配時，自動置中裁切
3. **無空白區**：確保容器完全填滿
4. **視覺統一**：所有圖片容器保持一致外觀

### ✅ 已調整檔案（18 個檔案）

#### 1. 首頁組件

## 🔧 技術細節

### 從 `object-contain` 改為 `object-cover object-center`：

| 屬性 | 作用 |
|------|------|
| `object-cover` | 圖片填滿容器，保持比例 |
| `object-center` | 裁切時從中心開始，保持平衡 |

### 例外情況（保留 `object-contain`）：

| 使用場景 | 原因 |
|---------|------|
| Logo 圖片 | 需要完整顯示品牌標誌 |
| 縮圖 | 需要預覽完整內容 |
| 燈箱圖片 | 需要查看完整圖片 |
